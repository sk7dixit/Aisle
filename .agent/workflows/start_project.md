# Start ShopLens Project

1. Start the Backend server
// turbo
Run `npx nodemon backend/server.js` in `s:\2K27_Project\shoplens`

2. Start the Frontend development server
// turbo
Run `npm run dev` in `s:\2K27_Project\shoplens\frontend`
